# Sarcopenia-Detection-L3-CT-Segmentation
Automatic assessment of skeletal muscle index (SMI) for sarcopenia diagnosis using U-Net segmentation in MATLAB App Designer.

### Project Description
The application utilizes a Deep Learning approach for the automated segmentation of skeletal muscles from CT medical images (transverse section at the L3 vertebral level).

The primary goal is to provide a rapid calculation of the **Skeletal Muscle Index (SMI)** to assist medical professionals in the clinical diagnosis of sarcopenia, particularly in oncological or surgical patients.

### Technologies & Tools
* **Development Environment:** MATLAB 
* **Toolboxes:** Deep Learning Toolbox, Image Processing Toolbox
* **Architecture:** U-Net 
* **Input Format:** CT Scans (.png / .jpg / .dcm)
